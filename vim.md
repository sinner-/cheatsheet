# vim Cheatsheet

* Convert file to UNIX format:
  * `:set ff=unix`
* Undo:
  * `u`
* Redo:
  * `Ctrl-r`
* Start of line:
  * `^`
* End of line:
  * `$`
* Insert to start of line:
  * `Shift-I`
* Insert to end of line:
  * `Shift-A`
* Visual block:
  * `Ctrl-v`
* Action against all lines:
  * `%`
  * e.g. `%d` deletes all lines.
* Write stdout from cmd into buffer:
  * `!r cmd`
